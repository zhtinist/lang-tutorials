Other Languages for C/C++ Programmers

Assuming you already know C/C++ — pointers, stack/heap, pass-by-value vs. reference, manual memory management.
These three tutorials skip the basic syntax and instead explain what actually happens under the hood for every operation, always contrasted with C/C++.

Python

Understand Python from the implementation up: memory model, object layout, concurrency, and the standard library — always contrasted with C/C++.

Java

Understand Java from the implementation up: memory model, object layout, concurrency, and the standard library — always contrasted with C/C++.

Go

Understand Go from the implementation up: memory model, object layout, concurrency, and the standard library — always contrasted with C/C++.

Why these three

  • Python — everything is a heap object, names are references, reference counting + GC, the GIL and free-threading.
  • Java — primitives vs. references, object headers and escape analysis, the JMM and virtual threads, GC evolution.
  • Go — everything is a value copy, the header structs behind slice/map/channel, escape analysis, the GMP scheduler, Swiss Tables.
